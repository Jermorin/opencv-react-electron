## Install

- install [server socket](https://github.com/Jermorin/opencv-server-socket)
- `npm install`

##Usage

- launch [server socket](https://github.com/Jermorin/opencv-server-socket)
- `npm run watch`
- `npm run start` or go to `http://localhost:8080/`
